Kebijakan Privasi

Siber Shop Store berkomitmen menjaga privasi dan kepercayaan pengguna melalui transparansi penuh. Kami menghargai setiap pengunjung dan menyampaikan bahwa:

  • Kami tidak menyimpan data pribadi apa pun dari pengguna.
  • Seluruh tools dan layanan bersifat open source dan dapat diakses secara publik.
  • Siber Shop mengandalkan API publik dan private yang terenkripsi dan aman.
Transparansi & Keamanan

Tidak ada pelacakan aktivitas pengguna, pengumpulan cookies untuk tujuan komersial, maupun pembagian informasi ke pihak ketiga. Semua proses dilakukan langsung di browser pengguna tanpa intervensi server kami.

Kode Terbuka

Kami mendorong kolaborasi melalui keterbukaan kode sumber. Siapa saja dapat memeriksa, mempelajari, dan mengembangkan ulang proyek ini sebagai bentuk transparansi dan edukasi.

Integrasi API

Beberapa fitur terhubung ke layanan pihak ketiga melalui API terenkripsi demi memastikan keakuratan dan kenyamanan pengguna. Data yang dimasukkan tidak akan disimpan atau dilacak.

Pembaruan Kebijakan

Kebijakan ini dapat diperbarui sewaktu-waktu. Perubahan besar akan diumumkan secara langsung di halaman ini.
