Kebijakan Privasi

Siber Shop Store berkomitmen untuk menjaga kepercayaan pengguna dengan transparansi penuh terkait privasi. Kami menghargai privasi setiap pengunjung dan ingin menegaskan bahwa:

  • Situs ini tidak menyimpan data pribadi apa pun dari pengguna.
  • Semua tools dan layanan yang tersedia bersifat open source dan dapat diakses publik.
  • Situs ini memanfaatkan API publik dan private dari internet yang telah dienkripsi dan diamankan secara ketat.
Transparansi & Keamanan

Kami tidak melakukan pelacakan, tidak mengumpulkan cookies untuk tujuan komersial, dan tidak membagikan informasi ke pihak ketiga. Setiap interaksi yang dilakukan pengguna berlangsung secara lokal di browser tanpa penyimpanan server dari pihak kami.

Kode Terbuka

Siber Shop didesain dengan prinsip keterbukaan. Kode sumber proyek ini dapat diperiksa, dipelajari, atau dikembangkan kembali oleh siapa saja. Kami percaya bahwa keterbukaan menciptakan transparansi dan kepercayaan.

Integrasi API

Beberapa fitur kami terhubung dengan API eksternal (baik publik maupun private) untuk memberikan hasil yang akurat dan fungsionalitas maksimal. Semua komunikasi dilakukan secara terenkripsi dan tidak menyimpan data input pengguna.

Pembaruan Kebijakan

Kebijakan privasi ini dapat diperbarui sewaktu-waktu. Perubahan signifikan akan diumumkan langsung di halaman ini.

Kontak Kami

Jika ada pertanyaan atau klarifikasi lebih lanjut, silakan hubungi kami melalui halaman Contact.
